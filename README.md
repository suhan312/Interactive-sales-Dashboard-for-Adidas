# Interactive-sales-Dashboard-for-Adidas
📊 Just Built an Interactive Sales Dashboard for Adidas! 🚀
I'm excited to share my latest data visualization project - an Adidas Interactive Sales Dashboard built with Python and Streamlit!
🎯 What it does:
This dashboard transforms complex sales data into actionable insights through interactive visualizations, helping stakeholders make data-driven decisions.
✨ Key Features:
✅ Total Sales by Retailer - Compare performance across Foot Locker, Walmart, Sports Direct, West Gear, Kohl's, and Amazon
✅ Sales Trends Over Time - Track monthly sales patterns and identify seasonal trends
✅ Geographic Analysis - Visualize sales performance and units sold across all US states
✅ Regional & City Breakdown - Interactive treemap showing sales distribution by region and city
🛠️ Tech Stack:
• Python
• Streamlit
• Plotly for interactive visualizations
• Pandas for data manipulation
💡 What I Learned:
Building this project enhanced my skills in creating user-friendly data applications and demonstrated how powerful Streamlit is for rapid dashboard development. One of the easiest ways to build interactive dashboards is with Streamlit
